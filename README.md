# tiny_sound_font

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     tiny_sound_font:
       github: your-github-user/tiny_sound_font
   ```

2. Run `shards install`

## Usage

```crystal
require "tiny_sound_font"
```
